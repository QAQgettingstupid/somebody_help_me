<span style="font-weight:bold;">　听说你们要参加国际外科学会在莱顿举办的研讨会，真是羡慕。</span>
<span style="font-weight:bold;">　如果要访问荷兰，最好先了解一下荷兰和日本的渊源。虽然只是我的一己之见，但还是把我知道的写下来。希望你们能在飞机上阅读，作为在荷兰的向导。</span>
<span style="font-weight:bold;">　莱顿是荷兰建国的圣地。十五世纪，现在的荷兰、比利时地区是哈布斯堡家族的领地。一五一六年，卡洛斯从外祖父费南多斯二世那里继承了西班牙国王的称号，一五一九年在与法兰西的法兰索瓦一世的继承权之争中获胜，成为神圣罗马帝国皇帝卡洛斯五世。他的统治领地包括现在的西班牙、荷兰、德国、意大利、奥地利、匈牙利，是东西夹攻法国的大帝国。</span>
<span style="font-weight:bold;">　位于广大神圣罗马帝国西北的荷兰渔业兴盛，十四世纪末发明了鲱鱼的储藏方法，开始出口，鲱鱼捕捞业蓬勃发展，国力增强。</span>
<span style="font-weight:bold;">　带来荷兰黄金时代的航海术和造船业，都是以鲱鱼捕捞业为母胎。因此荷兰有句谚语：「鲱鱼捕捞业是一切商业之母」。</span>
<span style="font-weight:bold;">　一五六八年，奥兰治公威廉一世（＝橙公威廉一世）指挥武装叛乱，揭开了以脱离西班牙为目标的八十年战争的序幕。</span>
<span style="font-weight:bold;">　一五七二年，毛织物业的城镇莱顿加入叛乱方，一五七三年被西班牙军包围。但一五七四年八月，威廉破坏河堤，用水攻包围西班牙军，十月解放了莱顿。这和羽柴秀吉用水攻包围备中高松城几乎是在同一时期。</span>
<span style="font-weight:bold;">　奥兰治公威廉一世赞赏市民勇敢的抵抗，询问莱顿市民要免除十年的征税，还是创立大学，莱顿市民选择了创立大学。</span>
<span style="font-weight:bold;">　一五七五年一月，莱顿大学创立的消息发布，同年二月八日开学。就这样，荷兰独立的圣地莱顿，成为了荷兰最古老的大学城。</span>
<span style="font-weight:bold;">　之后，荷兰独立战争分裂为承认新教徒的北部荷兰和天主教占优势的南部。一五七九年一月，南部的「阿拉斯同盟」承认了西班牙王的统治权。这相当于现在的比利时。几天后，北部七州的「乌特勒支同盟」决定继续与西班牙斗争。这相当于现在的荷兰。</span>
<span style="font-weight:bold;">　荷兰独立的八十年战争，是以从西班牙独立为目标的反西班牙战争。</span>
<span style="font-weight:bold;">　由于是以新教徒为中心，荷兰对其他宗教很宽容，缺乏天主教传教的狂热要素。这成为了吸引许多优秀学生来莱顿大学的理由。</span>
<span style="font-weight:bold;">　这也是采取锁国政策的德川幕府唯一承认的通商国。</span>
<span style="font-weight:bold;">　这个时期，荷兰也开始了与东洋的贸易。一五九五年四月，在阿姆斯特丹设立的航海公司于九七年八月从爪哇岛满载着东洋的商品回国，荷兰国内开始对东洋贸易的热情高涨。九八年六月，鹿特丹的贸易公司有五艘船向东洋出航。其中一艘在一六○○年四月，关原之战的半年前，漂到了丰后的佐志生。乘员的英国人威廉・亚当斯受到德川家康的赏识，被提拔为旗本，荷兰人雅各・范・赞德也在江户被赐予了宅邸。以他的名字命名的地名就是现在的东京八重洲。</span>
<span style="font-weight:bold;">　然而，东洋贸易过于混乱，利润很少。因此一六○二年三月，国策公司荷兰东印度公司（VOC）被设立。</span>
<span style="font-weight:bold;">　VOC被赋予了东洋贸易的独占权，与当地人缔结条约的权限，建造城寨，拥有军队的权利和国家特别赋予的保护。</span>
<span style="font-weight:bold;">　一六一九年，VOC侵略了雅加达，建造城寨并命名为巴达维亚，将该地作为商业中心，荷兰将爪哇的一部分作为殖民地。</span>
<span style="font-weight:bold;">　另一方面，一六○九年，荷兰东印度公司开设了平户商馆。</span>
<span style="font-weight:bold;">　当时，乘葡萄牙船来到日本的耶稣会，早一步进入了日本。那是为了对抗当时兴起的基督教而由天主教创立的激进传教团体，由于传教过于强硬而被德川幕府疏远，遭到放逐。</span>
<span style="font-weight:bold;">　德川幕府禁止天主教国家西班牙和葡萄牙入国，一六三九年，开始完全锁国。但是荷兰人是基督教，没有强行传教，所以幕府允许其交易。</span>
<span style="font-weight:bold;">　四一年六月，荷兰商馆奉幕府之命从平户移到长崎的出岛，此后荷兰独占了与日本的贸易。荷兰成为欧美的唯一交易国，对日本来说是侥幸。因为十八世纪，世界的医学中心从意大利转移到莱顿。赫曼・布隆哈特教授博得名声，是优秀的临床医生，同时也是启蒙家，他的门下辈出之后的格丁尼亚学派，爱丁堡学派和维也纳学派的始祖。</span>
<span style="font-weight:bold;">　十八世纪的医学，莱顿大学是最高学府。之后，医学的中心转移到德国，名门威兹堡大学的栋梁是西博德家族。家族的年轻冒险家、医生、博物学者菲利普・冯・西博德来到日本，对日本留下莫大的影响。</span>
<span style="font-weight:bold;">　他是德国人，却以荷兰商馆附设的医师身份来到日本。就像这样，日本和荷兰在医学发展上关系深厚，特别是莱顿是兰医学的发源地。</span>
<span style="font-weight:bold;">　你能造访荷兰，我真的很高兴。祈祷学会发表成功。一郎。</span>
